# youtube-live-2column-full-screen

https://files.rinsuki.net/2021/yt-pwc.html?v=WBfbkPTqUtU という上位互換が生まれたので供養ということで...

## Usage

1. インストールする。
2. 任意の Youtube Live を開く
3. 全画面表示する
4. ショートカットキー "1" を押すと右にコメント欄が表示される
